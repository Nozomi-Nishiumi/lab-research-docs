PART 01

VS Code を入れる

macOS

Apple Silicon / Intel 共通(Universal)

  1. code.visualstudio.com を開き、Download for macOS(Universal)を取得。
  2. ダウンロードした .zip を解凍し、Visual Studio Code.app を「アプリケーション」フォルダへドラッグ。
  3. 初回起動時、Gatekeeper の確認が出たら 「開く」。
  4. ターミナルから code . で開けるようにする:コマンドパレット ⇧⌘P →「Shell Command: Install 'code' command in PATH」を実行。

Windows

User Installer 推奨

  1. code.visualstudio.com を開き、Download for Windows を取得。
  2. インストーラを実行。「PATH への追加」にチェック(既定でON)。右クリック統合が欲しければ関連チェックも入れる。
  3. 完了後、VS Code を起動。
  4. ターミナル(PowerShell)から code . でフォルダを開けることを確認。

共通の初期設定

研究用途ならこの2つは最初に

  1. 日本語表示にするなら拡張機能で 「Japanese Language Pack」 を入れて再起動(任意)。
  2. Python を使うので 「Python」(Microsoft 製)拡張機能を入れる。仮想環境は venv か conda をプロジェクトごとに。
ヒント ── プロジェクトは ~/Developer のようなクラウド同期外のフォルダに置くと、ビルド生成物の同期衝突を避けられます。
PART 02

AI を連携する

どれも拡張機能ビュー ⇧⌘X / Ctrl+Shift+X から入ります。まず1つ選び、上限や使い勝手が合わなければ後から足す/切り替えでOK。

01
GitHub Copilot純正統合・カード不要で一番手軽
無料 2,000補完 / 50チャット・月
  1. 拡張機能ビューで 「GitHub Copilot」 を検索してインストール(Chat も同梱)。 ext install GitHub.copilot
  2. 右下の通知か左下のアカウントアイコンから GitHub アカウントでサインイン。
  3. ブラウザで Authorize → VS Code に自動で戻る。
  4. プランで Free を選ぶ(支払い情報不要)。適当な .py で補完が出れば完了。
注意 ── 月2,000補完+50チャットまで。よく書くと1週間ほどで上限。軽めの補完・質問向き。
02
Gemini Code Assist無料枠が寛大・大きめのコンテキスト
無料 個人向けに広め
  1. 拡張機能ビューで 「Gemini Code Assist」 を検索してインストール。 ext install Google.geminicodeassist
  2. サイドバーの Gemini アイコン →「Sign in / Get started」。
  3. Google アカウントでサインイン(個人利用は無料)し、無料利用に同意。
  4. 補完・チャットが有効になったことを確認。
注意 ── チャットで大きめのコンテキスト(約128kトークン)を扱えるので、補完を大量に使う場合に上限で困りにくい。
03
ClineClaude Code に近い自律エージェント・BYOK
拡張は無料 / モデルは別
  1. 拡張機能ビューで 「Cline」 を検索してインストール。 ext install saoudrizwan.claude-dev
  2. サイドバーの Cline アイコン を開く。
  3. モデルプロバイダを設定。実質無料なら:

    A) Google AI Studio の無料 Gemini API キーを貼る / B) ローカルの Ollama を繋ぐ(完全ローカル)

    ollama run qwen2.5-coder
  4. キーまたは接続先を保存し、タスクを日本語で入力。ファイル編集・コマンド実行は 毎回承認制。
注意 ── 拡張は無料、API 利用料は別。無料枠やローカルモデルなら実質ゼロ円。次章の解析コードを任せるならこの系統が向く。
