データを可視化する

R 本体と RStudio の導入から、実際にデータを図にするサンプルコードまでをまとめた資料です。可視化は base R 同梱の airquality データ(1973年ニューヨークの大気観測)を使い、ggplot2 で一通り描きます。

PART 01

R を入れる

RStudio はエディタで、R 本体は含みません。先に R を入れてから RStudio を入れます(RStudio は入っている R を自動で認識します)。

macOS

Apple Silicon / Intel でビルドが異なる

  1. cran.r-project.org → Download R for macOS。
  2. チップに合うビルドを選ぶ。Apple メニュー → このMacについてで、「チップ」が M 系なら arm64、「プロセッサ」に Intel とあれば Intel 版。
  3. ダウンロードした .pkg を実行し、指示どおり進める。
  4. グラフィックスの一部で必要になるので XQuartz も入れておく(任意)。

Windows

base ディストリビューション

  1. cran.r-project.org → Download R for Windows → base。
  2. Download R for Windows のインストーラを取得。
  3. 実行し、オプションは既定のまま進める。
  4. 完了。R のアイコンが出るが、以降は R を直接ではなく RStudio から使う。
PART 02

RStudio を入れる

RStudio Desktop(無料)

Posit 提供・オープンソース版は完全無料

  1. posit.co/download/rstudio-desktop を開く。ページが OS を判定し、適切なインストーラを案内する。
  2. macOS:.dmg を開き、RStudio を「アプリケーション」へドラッグ。Windows:.exe を実行し既定のまま進める。
  3. RStudio を起動。インストール済みの R を自動検出して、左下に R のバージョンが表示されれば成功。
補足 ── すでに VS Code を使っているなら、Posit の Positron(VS Code ベースで R と Python を1つで扱える IDE)も選択肢です。ただし R の学習・チュートリアルはほぼ RStudio 前提なので、まずは RStudio が無難です。
PART 03

パッケージと動作確認

可視化には ggplot2 を使います。tidyverse(ggplot2・dplyr などの詰め合わせ)をまとめて入れるのが簡単です。RStudio 左下の Console に次を貼って実行(Enter)。初回は1〜3分ほどかかります。

R · Console
# 初回のみ（インストール）
install.packages("tidyverse")

# 動作確認
library(ggplot2)
ggplot(mpg, aes(displ, hwy)) + geom_point()
つまずき対策 ── ダウンロードで止まる場合は CRAN ミラーを変更:options(repos = "https://cloud.r-project.org/") を実行してから再度インストール。権限エラーが出たら、個人ライブラリ作成の確認に「yes」で進めて問題ありません。
PART 04

可視化サンプル

新規スクリプト(⇧⌘N / Ctrl+Shift+N)に貼り付け、各行で ⌘⏎ / Ctrl+Enter を押すと実行され、右下の Plots ペインに図が出ます。まず下準備を実行してから、各図のコードを試してください。

下準備:パッケージ読み込みとデータ

airquality に日付列を足しておく(折れ線グラフ用)

R
library(ggplot2)
library(dplyr)

# airquality：1973年NYの大気観測（base R 同梱）
data(airquality)
aq <- airquality %>%
  mutate(Date = as.Date(paste(1973, Month, Day, sep = "-")))

1. ヒストグラム

1変数の分布。ここでは気温 Temp の頻度。

R
ggplot(aq, aes(x = Temp)) +
  geom_histogram(binwidth = 3, fill = "#1E6FD0", color = "white") +
  labs(title = "気温の分布", x = "気温 (°F)", y = "日数")

2. 棒グラフ

カテゴリ別の集計値。月別の平均オゾン濃度。

R
aq %>%
  group_by(Month) %>%
  summarise(Ozone_mean = mean(Ozone, na.rm = TRUE)) %>%
  ggplot(aes(x = factor(Month), y = Ozone_mean)) +
  geom_col(fill = "#12857A") +
  labs(title = "月別の平均オゾン濃度", x = "月", y = "平均オゾン (ppb)")

3. 折れ線グラフ

時系列などの連続的な推移。日付ごとの気温。

R
ggplot(aq, aes(x = Date, y = Temp)) +
  geom_line(color = "#E07B34") +
  labs(title = "気温の推移", x = "日付", y = "気温 (°F)")

4. 散布図

2変数の関係。気温とオゾン濃度。

R
ggplot(aq, aes(x = Temp, y = Ozone)) +
  geom_point(color = "#3B3E9E", alpha = 0.7) +
  labs(title = "気温とオゾンの関係", x = "気温 (°F)", y = "オゾン (ppb)")

5. 散布図 + 回帰直線

関係に直線をあてる。geom_smooth(method="lm")。

R
ggplot(aq, aes(x = Temp, y = Ozone)) +
  geom_point(alpha = 0.6) +
  geom_smooth(method = "lm", se = TRUE, color = "#8E3B5E") +
  labs(title = "回帰：オゾン ~ 気温", x = "気温 (°F)", y = "オゾン (ppb)")

# 回帰係数・決定係数の確認
model <- lm(Ozone ~ Temp, data = aq)
summary(model)

まとめて実行する

全体を1つのスクリプトに。新規 .R ファイルに貼って保存すれば再利用できる

R · sample-visualizations.R
# ── パッケージ（初回のみ install.packages("tidyverse")）
library(ggplot2)
library(dplyr)

# ── データ：airquality（1973年NYの大気観測・base R 同梱）
data(airquality)
aq <- airquality %>%
  mutate(Date = as.Date(paste(1973, Month, Day, sep = "-")))

# 1. ヒストグラム
ggplot(aq, aes(Temp)) +
  geom_histogram(binwidth = 3, fill = "#1E6FD0", color = "white") +
  labs(title = "気温の分布", x = "気温 (°F)", y = "日数")

# 2. 棒グラフ
aq %>% group_by(Month) %>%
  summarise(Ozone_mean = mean(Ozone, na.rm = TRUE)) %>%
  ggplot(aes(factor(Month), Ozone_mean)) +
  geom_col(fill = "#12857A") +
  labs(title = "月別の平均オゾン濃度", x = "月", y = "平均オゾン (ppb)")

# 3. 折れ線グラフ
ggplot(aq, aes(Date, Temp)) +
  geom_line(color = "#E07B34") +
  labs(title = "気温の推移", x = "日付", y = "気温 (°F)")

# 4. 散布図
ggplot(aq, aes(Temp, Ozone)) +
  geom_point(color = "#3B3E9E", alpha = 0.7) +
  labs(title = "気温とオゾンの関係", x = "気温 (°F)", y = "オゾン (ppb)")

# 5. 散布図 + 回帰直線
ggplot(aq, aes(Temp, Ozone)) +
  geom_point(alpha = 0.6) +
  geom_smooth(method = "lm", se = TRUE, color = "#8E3B5E") +
  labs(title = "回帰：オゾン ~ 気温", x = "気温 (°F)", y = "オゾン (ppb)")

summary(lm(Ozone ~ Temp, data = aq))

環境構築の資料の1つです。他の資料へは 資料一覧 から移動できます。可視化を自分のデータに置き換える場合は、aq を自分のデータフレームに、aes() の列名を自分の変数名に替えるだけで流用できます。
