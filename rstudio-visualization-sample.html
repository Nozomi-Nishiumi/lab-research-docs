別紙 · 可視化サンプル

データを可視化する
サンプルコード

RStudio 導入の別紙です。base R 同梱の airquality(1973年NYの大気観測)を使い、ggplot2 でヒストグラム・棒・折れ線・散布図・回帰線を、rgl で3次元散布図を描きます。新規スクリプト(⇧⌘N / Ctrl+Shift+N)に貼り、各行で ⌘⏎ / Ctrl+Enter を押すと右下の Plots ペインに図が出ます。各コードの下には実際に実行したときの出力(図・コンソール)を載せているので、手元の結果と見比べてください。まず下準備を実行してから各図を試します。

下準備:パッケージ読み込みとデータ

airquality に日付列を足しておく(折れ線グラフ用)。最後の head(aq) で中身を確認

R · INPUT
library(ggplot2)
library(dplyr)

# airquality：1973年NYの大気観測（base R 同梱）
data(airquality)
aq <- airquality %>%
  mutate(Date = as.Date(paste(1973, Month, Day, sep = "-")))

01

1. ヒストグラム

1変数の分布。ここでは気温 Temp の頻度。

R · INPUT
ggplot(aq, aes(x = Temp)) +
  geom_histogram(binwidth = 3, fill = "#1E6FD0", color = "white") +
  labs(title = "気温の分布", x = "気温 (°F)", y = "日数")
OUTPUT · Plots ペイン
読み方 ── 80°F(約27°C)付近の日が最も多く、左(低温側)に裾を引く分布。binwidth を変えると棒の幅(=階級幅)が変わるので、3 を 5 や 1 にして再実行してみてください。
02

2. 棒グラフ

カテゴリ別の集計値。月別の平均オゾン濃度。

R · INPUT
aq %>%
  group_by(Month) %>%
  summarise(Ozone_mean = mean(Ozone, na.rm = TRUE)) %>%
  ggplot(aes(x = factor(Month), y = Ozone_mean)) +
  geom_col(fill = "#12857A") +
  labs(title = "月別の平均オゾン濃度", x = "月", y = "平均オゾン (ppb)")
OUTPUT · Plots ペイン
読み方 ── 真夏(7・8月)の平均オゾンが約60ppbと突出し、5月の2倍以上。na.rm = TRUE は「NA を除いて平均する」指定で、これを外すと NA を含む月の平均が NA になってしまいます。
03

3. 折れ線グラフ

時系列などの連続的な推移。日付ごとの気温。

R · INPUT
ggplot(aq, aes(x = Date, y = Temp)) +
  geom_line(color = "#E07B34") +
  labs(title = "気温の推移", x = "日付", y = "気温 (°F)")
OUTPUT · Plots ペイン
読み方 ── 5月から夏に向けて上昇し、8月末に97°Fのピーク。下準備で作った Date 列(Date型)を x に使っているので、軸が「May, Jun, …」と月表示になります。Month と Day のままでは連続した時間軸になりません。

4. 散布図

2変数の関係。気温とオゾン濃度。

R · INPUT
ggplot(aq, aes(x = Temp, y = Ozone)) +
  geom_point(color = "#3B3E9E", alpha = 0.7) +
  labs(title = "気温とオゾンの関係", x = "気温 (°F)", y = "オゾン (ppb)")
OUTPUT · Plots ペイン
OUTPUT · Console(警告)
Warning message:
Removed 37 rows containing missing values or values outside the scale
range (`geom_point()`).
この警告は正常 ── Ozone が NA の37日分は点が描けないため自動で除外された、という通知でエラーではありません。図はそのまま使えます。気温が高いほどオゾンも高い「正の関係」が見えます(alpha = 0.7 は点の透過で、重なりを見やすくする指定)。

5. 散布図 + 回帰直線

関係に直線をあてる。geom_smooth(method="lm")。

R · INPUT
ggplot(aq, aes(x = Temp, y = Ozone)) +
  geom_point(alpha = 0.6) +
  geom_smooth(method = "lm", se = TRUE, color = "#8E3B5E") +
  labs(title = "回帰：オゾン ~ 気温", x = "気温 (°F)", y = "オゾン (ppb)")
OUTPUT · Plots ペイン
読み方 ── 直線が回帰直線、グレーの帯が95%信頼区間(se = TRUE)。続けて数値でも確認します。
R · INPUT
# 回帰係数・決定係数の確認

summary(model)

6. 3次元散布図(rgl)

3変数の関係を立体で。気温 × 風速 × オゾン。マウスで回せる対話的3D。

R · INPUT
# 初回のみ（インストール）
install.packages("rgl")

library(rgl)

# 3次元散布図：x=気温, y=風速, z=オゾン
plot3d(aq$Temp, aq$Wind, aq$Ozone,
       type = "s", size = 1, col = "#3B3E9E",
       xlab = "Temp", ylab = "Wind", zlab = "Ozone")
OUTPUT · rgl ウィンドウ(スナップショット)
ここが2Dと違う ── 実行すると Plots ペインではなく別ウィンドウに対話的な3Dビューが開きます。ドラッグで回転、ホイールでズーム。上の画像は見やすい向きに回してからのスナップショットなので、開いた直後の向きとは違って見えても正常です。type = "s" は点を球で描く指定。高温(Temp大)かつ弱風(Wind小)の隅でオゾンが高く盛り上がるのが回すとよく分かります。
R · INPUT
# 重回帰 Ozone ~ Temp + Wind の回帰平面を重ねる
model3d <- lm(Ozone ~ Temp + Wind, data = aq)
cf <- coef(model3d)
planes3d(cf["Temp"], cf["Wind"], -1, cf["(Intercept)"],
         alpha = 0.4, col = "#C2557F")

summary(model3d)

まとめて実行する

全体を1つのスクリプトに。新規 .R ファイルに貼って保存すれば再利用できる

R · sample-visualizations.R
# ── パッケージ（初回のみ install.packages("tidyverse")）
library(ggplot2)
library(dplyr)

# ── データ：airquality（1973年NYの大気観測・base R 同梱）
data(airquality)
aq <- airquality %>%
  mutate(Date = as.Date(paste(1973, Month, Day, sep = "-")))

# 1. ヒストグラム
ggplot(aq, aes(Temp)) +
  geom_histogram(binwidth = 3, fill = "#1E6FD0", color = "white") +
  labs(title = "気温の分布", x = "気温 (°F)", y = "日数")

# 2. 棒グラフ
aq %>% group_by(Month) %>%
  summarise(Ozone_mean = mean(Ozone, na.rm = TRUE)) %>%
  ggplot(aes(factor(Month), Ozone_mean)) +
  geom_col(fill = "#12857A") +
  labs(title = "月別の平均オゾン濃度", x = "月", y = "平均オゾン (ppb)")

# 3. 折れ線グラフ
ggplot(aq, aes(Date, Temp)) +
  geom_line(color = "#E07B34") +
  labs(title = "気温の推移", x = "日付", y = "気温 (°F)")

# 4. 散布図
ggplot(aq, aes(Temp, Ozone)) +
  geom_point(color = "#3B3E9E", alpha = 0.7) +
  labs(title = "気温とオゾンの関係", x = "気温 (°F)", y = "オゾン (ppb)")

# 5. 散布図 + 回帰直線
ggplot(aq, aes(Temp, Ozone)) +
  geom_point(alpha = 0.6) +
  geom_smooth(method = "lm", se = TRUE, color = "#8E3B5E") +
  labs(title = "回帰：オゾン ~ 気温", x = "気温 (°F)", y = "オゾン (ppb)")

summary(lm(Ozone ~ Temp, data = aq))

# 6. 3次元散布図（rgl・初回のみ install.packages("rgl")）
library(rgl)
plot3d(aq$Temp, aq$Wind, aq$Ozone,
       type = "s", size = 1, col = "#3B3E9E",
       xlab = "Temp", ylab = "Wind", zlab = "Ozone")

model3d <- lm(Ozone ~ Temp + Wind, data = aq)
cf <- coef(model3d)
planes3d(cf["Temp"], cf["Wind"], -1, cf["(Intercept)"],
         alpha = 0.4, col = "#C2557F")
summary(model3d)
実行環境 ── このページの出力は R 4.4.1 + ggplot2 / rgl で実際に実行して得たものです。バージョンが違っても図の見た目(フォント・配色の細部)以外は同じ結果になります。

可視化を自分のデータに置き換えるときは、aq を自分のデータフレームに、aes() の列名を自分の変数名に替えるだけで流用できます。RStudio 導入へ戻る。
